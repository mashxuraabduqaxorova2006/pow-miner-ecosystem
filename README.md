# ⛏️ PoW Miner Ecosystem

**PoW Miner Ecosystem** — bu Proof-of-Work (PoW) algoritmi asosida ishlovchi, mukammal vizualizatsiyaga ega bo'lgan Web3 mayning simulyatori. O'yinda siz bloklarni minalashingiz, xeshlar zanjirini kuzatishingiz va yutgan tokenlaringizni boshqa aktivlarga (ETH, USD, BTC) almashtirishingiz mumkin.

## 🚀 O'yin Xususiyatlari
- **⛓️ Haqiqiy PoW Mexanikasi**: SHA-256 xeshlash algoritmi orqali bloklarni qidirish va minalash.
- **🔗 Immutable Ledger**: Barcha minalangan bloklar zanjir shaklida bog'langan va brauzer xotirasida saqlanadi.
- **🔄 Multi-Asset Swap**: MNT tokenlarini real vaqtda ETH, USD yoki BTC-ga almashtirish imkoniyati.
- **📜 Amallar Tarixi**: Barcha mining va swap harakatlaringizni kuzatib borish uchun maxsus panel.
- **🦊 MetaMask Integratsiyasi**: Web3 hamyon orqali to'liq nazorat va xavfsiz tranzaksiyalar.
- **🌌 Premium Dizayn**: Cyberpunk uslubidagi interfeys, neon effektlar va silliq animatsiyalar.

## ✨ Texnologik Stack
- **Frontend**: React.js, Vite
- **Blockchain**: Solidity, Hardhat, Ethers.js
- **Styling**: Vanilla CSS (Premium Glassmorphism)

## 🕹 Qanday O'ynash Kerak?
1. **Hamyonni Ulang**: Sahifadagi "MetaMask-ni ulash" tugmasini bosing.
2. **Miningni Boshlang**: "Miningni boshlash" tugmasini bosing va kompyuteringiz xesh qidirishini kuzating.
3. **Mukofot Oling**: Har bir minalangan blok uchun sizga 10 MNT mukofot beriladi.
4. **Almashtiring (Swap)**: To'plangan tokenlarni o'zingizga qulay valyutaga (ETH, USD yoki BTC) almashtiring.
5. **Tarixni Kuzating**: Amallar tarixi bo'limida barcha yutuqlaringizni ko'rib boring.
